Fast, single-binary WiFi security auditing

Capture WPA handshakes, inject deauth frames, and crack passwords — all from one 6 MB executable with zero third-party dependencies.

// Features

Everything runs natively in Go. No Python, no aircrack-ng, no pip install.

[>] scan

Passive 802.11 Scanning

Parses raw beacons, probes, and data frames via gopacket. Discovers ESSIDs, encryption, WPS, clients, and signal strength in real time.

[>] capture

WPA Handshake Capture

Native pcap writer with live EAPOL tracking. Monitors 4-way handshake state per client and fires the moment M1+M2 are captured.

[>] inject

Deauth Injection

Constructs and transmits raw deauthentication frames via gopacket layer serialization and pcap_sendpacket. No aireplay-ng needed.

[>] crack

Parallel Dictionary Cracker

Native PBKDF2-SHA1 with full PMK→PTK→MIC verification. Distributes work across all CPU cores via goroutines. ~200-400 keys/sec per core.

[>] tui

Interactive TUI

Full-screen terminal interface built with Bubble Tea. Live-updating scan table, attack progress view, results browser, and vim-style navigation.

[>] zero-dep

Zero Dependencies

Core functionality requires only iw, ip, and libpcap — standard on every Linux distro. Single static binary, no runtime interpreter.

// Install

Download the binary and run. That's it.

download & run 
# Download the latest release
$ curl -LO https://github.com/jimmytoenners/wifibear/releases/latest/download/wifibear-linux-amd64

# Make it executable
$ chmod +x wifibear-linux-amd64

# Run it
$ sudo ./wifibear-linux-amd64
build from source 
$ git clone https://github.com/jimmytoenners/wifibear.git
$ cd wifibear
$ make build
$ sudo ./wifibear
// Attack Chain

Select a WPA/WPA2 target and WifiBear runs the optimal attack sequence.

1

PMKID Capture

Fast, clientless attack. Captures the PMKID from the AP's first EAPOL frame. 30s timeout, skipped if hcxdumptool unavailable.

optional — hcxdumptool
2

WPA 4-Way Handshake

Opens pcap capture on the target channel. Injects deauth frames every 15s to force client reconnections. Monitors EAPOL state in real time — stops the instant a complete handshake is captured.

native — zero dependencies
3

Dictionary Crack

PBKDF2-SHA1 with 4096 iterations per passphrase. Full PMK → PTK → MIC verification pipeline parallelized across all CPU cores.

native — zero dependencies
// CLI Reference

Key flags and options.

Flag Default Description
-i, --interface auto Wireless interface to use
-w, --wordlist rockyou.txt Path to wordlist for cracking
--pillage off Attack all targets automatically
--bssid — Target a specific access point
--essid — Target a specific network name
--no-pmkid off Skip PMKID, go straight to handshake
--wpa-only off Skip WPS and WEP attacks
--scan-timeout 30s Scan duration before target selection
--hs-timeout 8m20s Handshake capture timeout
--deauth-interval 15s Time between deauth rounds
--mac-randomize on Randomize MAC address on start
-v, --verbose 1 Verbosity level (0-3)
